🎮 Duck Hunt – Canvas 2D

Videojuego desarrollado en HTML5, CSS y JavaScript utilizando Canvas 2D.
El objetivo es disparar a los patos y cumplir la meta obligatoria de cada nivel para avanzar hasta el nivel 10.

🚀 Características

10 niveles progresivos

Meta obligatoria por nivel (Nivel 1 = 3 hasta Nivel 10 = 12)

Sistema de balas y recarga

Sonidos estilo 8-bit

Animación del perro caminando

Sistema de puntaje y récord guardado

🎮 Controles

Click → Disparar

R → Recargar

P → Pausar

M → Activar/Desactivar sonido

🛠️ Tecnologías utilizadas

HTML5

CSS3 + Bootstrap

JavaScript

Canvas 2D API

Web Audio API
